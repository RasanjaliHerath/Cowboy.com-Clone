# 🏍️ Cowboy.com Clone

A modern and responsive clone of the Cowboy.com landing page built using **React**, **Vite**, **Tailwind CSS**, and **Material UI**. This project replicates the sleek design and smooth user experience of the original site, including hero video backgrounds, bold typography, and responsive layouts.


## 🛠️ Tech Stack

| Tech             | Description                        |
|------------------|------------------------------------|
| React            | JavaScript library for UI building |
| Vite             | Blazing-fast frontend tooling       |
| Tailwind CSS     | Utility-first CSS framework         |
| Material UI      | React UI component library          |
